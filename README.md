# ArduinoClock
![](https://github.com/tailer954/ArduinoClock/blob/main/ElectricalClock.gif)
## 概述
主要利用微控制器 Arduino Uno 和一個 20 乘 4 大小的液晶螢幕，製作一個能顯示星期、日期、時間的電子鐘
    
## 使用方式
需要下載 Arduino IDE 後，於右方由 johnrickman 提供的 LCD 函式庫載點 [載點](https://github.com/johnrickman/LiquidCrystal_I2C) 將 LiquidCrystal_I2C.h 加入 Arduino 的函式庫中，方能將程式碼 ElectricalClock.ino 順利地燒入 Arduino Uno
